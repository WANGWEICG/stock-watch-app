# 兄弟看盤｜台股看盤軟體

一個可直接放到 GitHub Pages 的台股看盤網頁 App。

## 功能

- 自選股清單：預設 0050、0056、2330
- 折線圖：收盤價走勢
- 指標：KD、RSI、MA5、MA20
- 價格警示：高於 / 低於指定價格
- KD-K 小於 20 提醒
- localStorage 保存自選股與警示
- 支援手機瀏覽，可加入主畫面

## 使用方式

1. 把所有檔案上傳到 GitHub Repository。
2. 到 Settings → Pages。
3. Source 選 main branch，資料夾選 root。
4. 等 GitHub Pages 產生網址後，用手機打開即可。

## 資料說明

- 預設為示範資料，避免 CORS 或非交易日造成畫面空白。
- 按「切換示範資料」後，會嘗試抓 TWSE 月成交資訊。
- 真正即時報價、背景推播、券商下單，需要後端伺服器與合法資料源/API。
