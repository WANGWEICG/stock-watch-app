# 兄弟看盤 Pro

手機可用的 GitHub Pages 股市看盤網頁 App。

## 功能
- 0050、0056、TSLA、SPY、QQQ 預設看盤
- 自選股：台股代號 / 美股代號
- 讀取公開行情資料
- MA5、MA20、RSI、KD 計算
- 0050 K值小於20提醒
- 每 60 秒自動更新

## 上傳 GitHub Pages
1. 解壓縮 ZIP
2. 把 `index.html`、`style.css`、`app.js`、`manifest.json`、`icon.svg`、`README.md` 全部上傳到 Repository 根目錄
3. Settings → Pages → Deploy from branch → main / root
